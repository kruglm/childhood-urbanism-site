# Детская среда — MVP сайта

Статический сайт для проекта «Индекс детской урбанистики России / Childhood Urbanism Index».

## Как запустить локально для дебага

```bash
cd childhood-urbanism-site
python3 -m http.server 8000
```

Потом открыть в браузере:

```text
http://localhost:8000
```

## Как опубликовать на GitHub Pages

1. Создать репозиторий на GitHub.
2. Загрузить в него файлы `index.html`, `styles.css`, `app.js`, `.nojekyll` и папку `assets`.
3. Открыть `Settings → Pages`.
4. В `Build and deployment` выбрать `Deploy from a branch`.
5. Выбрать ветку `main` и папку `/root`.
6. Сохранить. Через пару минут сайт будет доступен по ссылке GitHub Pages.

## Где менять контент

Основной контент сейчас хранится в `app.js`:

- массив `industry` — карточки организаций;
- массив `education` — образовательные инициативы;
- массив `knowledge` — темы базы знаний.

Статичные тексты секций находятся в `index.html`.

## Шрифты

В CSS уже прописаны названия `Vela Sans` и `CrazyV1` как предпочтительные шрифты, но сами файлы шрифтов не включены в архив. Если у тебя есть лицензированные файлы шрифтов, положи их в `assets/fonts/` и добавь в начало `styles.css`:

```css
@font-face {
  font-family: "Vela Sans";
  src: url("assets/fonts/VelaSans-Regular.woff2") format("woff2");
  font-weight: 400 800;
}

@font-face {
  font-family: "CrazyV1";
  src: url("assets/fonts/CrazyV1.woff2") format("woff2");
}
```

Если файлов нет, сайт всё равно будет выглядеть нормально на запасных шрифтах.
